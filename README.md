# Erlang_C
Проект содержит два приложения, серверное на С с использованием ZMQ Publisher и клиентское на Erlang с использованием ZMQ Subsriber Chumak
Для работы проекта нужно наличие zmq, rebar3, erlang.

Установка zmq:
sudo apt-get install libzmq3-dev

Установка Erlang:
sudo apt-get install erlang

Установка Rebar3:
sudo apt-get install rebar3

Для запуска проекта нужно выполнить следующие шаги:

ШАГ 1
Введите команду make в терминале корневой папк. В ответ вы получите список доступных команд:
	"make server    - Компиляция и запуск сервера"
	"make client    - Компиляция и запуск клиента"
	"make clean     - Очистка скомпилированных файлов"

ШАГ 2
Запустите сервер командой make server

ШАГ3 
Откройте еще один терминал из корневой папки и запустите клиента командой make client


Для тестирования проекта в терминале с запущенным серверным приложением нужно вводить сообщение, после чего на терминале с запущенным клиентским приложением будет приходить данное сообщение.
