# Erlang_C
Проект содержит два приложения, серверное на С с использованием ZMQ Publisher и клиентское на Erlang с использованием ZMQ Subsriber Chumak
Для работы проекта нужно наличие zmq, rebar3, erlang.

Установка zmq:
sudo apt-get install libzmq3-dev

Установка Erlang:
sudo apt-get install erlang

Установка Rebar3:
sudo apt-get install rebar3

Для запуска проекта нужно выполнить следующие шаги:

ШАГ 1
Запустить серверное приложение, для этого нужно открыть терминал из папки server.
В терминале ввести команду make, приложение соберется и выдаст сообщение в терминале "Введите сообщение:"

ШАГ 2
Запустить клиенское приложение, для этого нужно открыть терминал из папки client.
В терминале ввести каманду make, в терминал выведется список команд для работы с приложением.

Доступные команды:
"make compile- Компиляция проекта"
"make run - Запуск"

ШАГ 3 
Скомпилировать клиентское приложение, для этого ввести в терминал make compile.

ШАГ4
Запустить клиентское приложение, для этого ввести в терминале make run.


Для тестирования проекта в терминале с запущенным серверным приложением нужно вводить сообщение, после чего на терминале с запущенным клиентским приложением будет приходить данное сообщение.
